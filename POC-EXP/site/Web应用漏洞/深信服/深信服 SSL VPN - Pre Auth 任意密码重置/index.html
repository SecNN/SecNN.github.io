二、漏洞影响¶

高版本(如M7.6.8R2) 直接删除相关函数

漏洞分析¶

差不多的逻辑

唯独多了个 RC4 解密，key 是 20100720

在数据提取中写的有点奇怪,使用,和=作为分隔符，所以我们的数据也要类似如：

,username=test,ip=127.0.0.1,grpid=1,pripsw=suiyi,newpsw=QQ123456,

M7.6.6R1 key 为 20181118

M7.6.1 key 为 20100720
