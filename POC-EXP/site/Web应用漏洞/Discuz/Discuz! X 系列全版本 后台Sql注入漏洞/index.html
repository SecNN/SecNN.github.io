Discuz! X 系列全版本 后台Sql注入漏洞¶

一、漏洞简介¶

利用条件：

1.知道网站的绝对路径

2.secure_file_priv的值为空

二、漏洞影响¶

Discuz!X 系列全版本 截止到 Discuz!X 3.4 R20191201 UTF-8

三、复现过程¶

挖过discuz 漏洞的都知道 它会对大部分传参进来的值进行过滤和校验 ，所以当时找了一个二次注入的点
