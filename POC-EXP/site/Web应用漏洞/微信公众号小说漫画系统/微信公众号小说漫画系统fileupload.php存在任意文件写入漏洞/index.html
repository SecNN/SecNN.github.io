微信公众号小说漫画系统fileupload.php存在任意文件写入漏洞¶

一、漏洞简介¶

微信公众号小说漫画系统前台任意文件写入漏洞，允许攻击者上传恶意文件到服务器，可能导致远程代码执行、网站篡改或其他形式的攻击，严重威胁系统和数据安全。

二、影响版本¶

  • 微信公众号小说漫画系统

三、资产测绘¶

  • fofa"/Public/home/mhjs/jquery.js"
  • 特征
