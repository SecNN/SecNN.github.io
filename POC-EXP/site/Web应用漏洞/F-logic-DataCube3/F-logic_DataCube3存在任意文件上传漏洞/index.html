F-logic_DataCube3存在任意文件上传漏洞¶

F-logic DataCube3 /admin/setting_photo.php接口处存在任意文件上传漏洞，恶意攻击者可通过该漏洞在服务器端写入后门，获取服务器权限，进而控制整个web服务器。
