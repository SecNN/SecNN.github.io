由此可知，normalize()函数对经过经过parsePathParameters()函数过滤过;xxx/的URL请求内容进标准化处理，具体为将连续的多个/给删除掉只保留一个、将/./删除掉、将/../进行跨目录拼接处理，最后返回处理后的URL路径。

结论

Tomcat对/;xxx/以及/./的处理是包容的、对/../会进行跨目录拼接处理。
