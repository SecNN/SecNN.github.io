Crawlab users 任意用户添加漏洞¶

漏洞描述¶

Crawlab users 的 api 存在任意用户添加，且添加为未授权接口，可通过添加后在后台进一步攻击

漏洞影响¶

Crawlab v0.0.1

网络测绘¶
