金山 V8 终端安全系统 pdf_maker.php 命令执行漏洞¶

漏洞描述¶

金山 V8 终端安全系统 pdf_maker.php 存在命令执行漏洞，由于没有过滤危险字符，导致构造特殊字符即可进行命令拼接执行任意命令

漏洞影响¶

金山 V8 终端安全系统

网络测绘¶

app="猎鹰安全-金山V8+终端安全系统"
