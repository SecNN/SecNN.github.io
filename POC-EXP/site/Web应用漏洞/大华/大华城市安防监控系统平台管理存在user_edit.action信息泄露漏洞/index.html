大华城市安防监控系统平台管理存在user_edit.action信息泄露漏洞¶

大华DSS城市安防监控平台是一个在通用安防视频监控系统基础上设计开发的系统。该平台user_edit.action泄露了敏感信息漏洞，攻击者可以通过此漏洞获取管理员对应权限。
