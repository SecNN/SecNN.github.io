大华 DSS 视频管理系统attachment_clearTempFile存在SQL注入漏洞¶

一、漏洞简介¶

大华DSS城市安防监控平台是一个在通用安防视频监控系统基础上设计开发的系统。大华DSS城市安防监控平台attachment_clearTempFile文件存在SQL注入漏洞，攻击者可以通过此漏洞获取数据库权限。

二、影响版本¶

  • 大华 DSS 视频管理系统

三、资产测绘¶

  • hunter：app.name=="Dahua 大华 DSS 视频管理系统"

  • 登录页面
