大华智能物联综合管理平台justForTest用户登录漏洞¶

浙江大华技术股份有限公司智能物联综合管理平台 用户登录接口/evo-apigw/evo-oauth/oauth/token存在漏洞，使用用户justForTest/任意密码即可成功登录平台，造成信息泄露。

Content-Length: 109
Content-Type: application/x-www-form-urlencoded
