EasyImage down.php 任意文件读取漏洞¶

一、漏洞简介¶

EasyImage：一个简洁的开源图床程序，支持多文件上传,简单无数据库,返回图片url,markdown,bbscode,html的一款图床程序。EasyImage down.php处存在任意文件读取漏洞。

二、影响版本¶

  • EasyImage

三、资产测绘¶

  • fofaapp="EasyImage-简单图床"
  • 特征
