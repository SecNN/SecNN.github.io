一、漏洞简介¶

先锋WEB燃气收费系统是由杭州先锋电子技术股份有限公司开发的一款服务于能源行业的系统，先锋WEB燃气收费系统存在文件上传漏洞，可导致攻击者获取服务器权限。

二、影响版本¶

  • 先锋WEB燃气收费系统
