智邦国际ERP GetPersonalSealData.ashx接口SQL注入漏洞¶

一、漏洞复现¶

智邦国际ERP系统 GetPersonalSealData.ashx接口处存在SQL注入漏洞，未经身份认证的攻击者可利用此漏洞获取数据库敏感信息，深入利用可获取服务器权限。

二、影响版本¶

  • 智邦国际ERP
