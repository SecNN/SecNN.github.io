Gerapy parse 后台远程命令执行漏洞¶

漏洞描述¶

Gerapy gerapy/server/core/views.py 中的方法存在任意命令执行，攻击者登录后台后发送特定的请求包即可利用漏洞

漏洞影响¶

Gerapy <= 0.9.7

网络测绘¶

title="Gerapy"

漏洞复现¶

登录页面

出现漏洞的文件为 gerapy/server/core/views.py
