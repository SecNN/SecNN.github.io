SuiteCRM responseEntryPoint存在SQL注入漏洞¶

一、漏洞简介¶

SuiteCRM存在SQL注入漏洞，未经身份验证的远程攻击者可以通过该漏洞拼接执行SQL注入语句，从而获取数据库敏感信息。

二、影响版本¶

  • SuiteCRM

三、资产测绘¶

title="SuiteCRM"
