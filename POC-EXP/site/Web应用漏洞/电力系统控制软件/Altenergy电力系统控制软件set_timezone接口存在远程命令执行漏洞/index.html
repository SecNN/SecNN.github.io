Altenergy电力系统控制软件set_timezone接口存在远程命令执行漏洞¶

电力系统控制软件 Altenergy Power Control Software C1.2.5版本的系统/set_timezone接口存在命令注入漏洞，攻击者可执行任意命令获取服务器权限。
