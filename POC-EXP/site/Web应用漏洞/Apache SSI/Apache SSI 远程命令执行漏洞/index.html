Apache SSI 远程命令执行漏洞¶

一、漏洞简介¶

在测试任意文件上传漏洞的时候，目标服务端可能不允许上传php后缀的文件。如果目标服务器开启了SSI与CGI支持，我们可以上传一个shtml文件，并利用``语法执行任意命令。

三、复现过程¶
