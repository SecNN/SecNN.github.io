启明星辰天玥运维安全网关tagid参数存在SQL注入漏洞¶

一、漏洞简介¶

天玥网络安全审计系统是针对业务环境下用户对网络内的核心IT资产和服务器进行的操作行为进行细粒度审计的合规性管理系统。 启明星辰天玥网络安全审计系统tagid参数存在SQL注入漏洞，攻击者可利用该漏洞获取数据库敏感信息。

二、影响版本¶

  • 天玥运维安全网关

三、资产测绘¶

  • hunterapp.name="启明星辰天玥运维安全网关"
  • 特征
