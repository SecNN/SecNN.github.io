AJ-Report开源数据大屏存在远程命令执行漏洞¶

该平台可以通过post方式在validationRules参数对应值中进行命令执行，可以获得服务器权限，登陆管理后台接管大屏。如果被不法分子利用，书写反动标语，危害后果十分严重。

影响版本¶

最新版本v1.4.0
