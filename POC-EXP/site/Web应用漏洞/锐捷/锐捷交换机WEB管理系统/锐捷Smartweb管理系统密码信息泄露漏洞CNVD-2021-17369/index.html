锐捷Smartweb管理系统 密码信息泄露漏洞 CNVD-2021-17369¶

一、漏洞简介
锐捷网络股份有限公司无线smartweb管理系统存在逻辑缺陷漏洞，攻击者可从漏洞获取到管理员账号密码，从而以管理员权限登录。
二、影响版本
锐捷网络股份有限公司 无线smartweb管理系统
三、资产测绘

●登录页面

使用默认口令guest/guest登录系统

之后访问

/web/xml/webuser-auth.xml
