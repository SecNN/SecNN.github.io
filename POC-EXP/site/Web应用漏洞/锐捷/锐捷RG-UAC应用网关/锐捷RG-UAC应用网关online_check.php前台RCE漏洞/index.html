锐捷RG-UAC应用网关online_check.php前台RCE漏洞¶

一、漏洞描述¶

锐捷RG-UAC应用管理网关 online_check.php 接口处存在命令执行漏洞，未经身份认证的攻击者可执行任意命令控制服务器权限。

二、影响版本¶

锐捷RG-UAC应用网关

三、资产测绘¶

fofa：app="Ruijie-RG-UAC"

特征：
