东胜物流软件SaveUserQuerySetting存在SQL注入漏洞¶

一、漏洞简介¶

东胜物流软件是一款致力于为客户提供IT支撑的 SOP， 帮助客户大幅提高工作效率，降低各个环节潜在风险的物流软件。东胜物流软件 SaveUserQuerySetting接口处存在 SQL 注入漏洞，攻击者可通过该漏洞获取数据库敏感信息。

二、影响版本¶

  • 东胜物流软件
