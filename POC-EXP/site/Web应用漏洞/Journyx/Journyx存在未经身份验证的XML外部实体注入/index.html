Journyx存在未经身份验证的XML外部实体注入¶

Journyx项目管理软件 soap_cgi.pyc 接口存在XML实体注入漏洞，未经身份认证的攻击者可以利用此漏洞读取系统内部敏感文件，获取敏感信息，使系统处于极不安全的状态。

"Journyx"

User-Agent: curl/8.1.2
