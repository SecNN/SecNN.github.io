好视通视频会议系统存在任意文件读取漏洞¶

一、漏洞简介¶

好视通视频会议是由深圳市华视瑞通信息技术有限公司开发，其在国内率先推出了3G互联网视频会议，并成功应用于SAAS领域。好视通视频会议系统存在任意文件读取漏洞，攻击者可通过该漏洞获取敏感信息。

二、影响版本¶

  • 好视通视频会议系统

三、资产测绘¶

  • hunterapp.name="好视通 Server Management System"
  • 特征
