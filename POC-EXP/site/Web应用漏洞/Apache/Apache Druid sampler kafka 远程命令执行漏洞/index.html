Apache Druid sampler kafka 远程命令执行漏洞¶

漏洞描述¶

Apache Druid 支持从 Kafka 加载数据，恶意的攻击者可通过修改 Kafka 连接配置属性，从而进一步触发 JNDI 注入攻击，最终攻击者可在服务端执行任意恶意代码，获取系统服务权限。

漏洞影响¶

Apache Druid

网络测绘¶

title_string = "Apache Druid"
