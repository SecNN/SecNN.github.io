Zkteco百傲瑞达安防管理系统平台存在shiro反序列化漏洞¶

一、漏洞简介¶

Zkteco 百傲瑞达安防管理系统平台存在 shiro 反序列化漏洞，该漏洞源于软件存在硬编码的 shiro-key，攻击者可利用该 key 生成恶意的序列化数据，在服务器上执行任意代码，执行系统命令、或打入内存马等，获取服务器权限。

二、影响版本¶

  • Zkteco 百傲瑞达安防管理系统平台
