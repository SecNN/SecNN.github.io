驰骋BPM RunSQL_Init存在SQL注入漏洞¶

一、漏洞简介¶

驰骋BPM是一款功能强大的业务流程管理平台，可提供可视化的设计界面和灵活的流程配置，帮助企业轻松构建、管理和优化各类业务流程。驰骋BPM RunSQL_Init存在SQL注入漏洞，攻击者可通过该漏洞获取账号密码。

二、影响版本¶
