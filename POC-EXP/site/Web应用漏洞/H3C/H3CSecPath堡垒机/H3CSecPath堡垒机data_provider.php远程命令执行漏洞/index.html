H3C SecPath堡垒机 data_provider.php 远程命令执行漏洞¶

一、漏洞简介¶

H3C SecParh堡垒机 data_provider.php 存在远程命令执行漏洞，攻击者通过任意用户登录或者账号密码进入后台就可以构造特殊的请求执行命令。

二、影响版本¶

  • H3C SecParh堡垒机

三、资产测绘¶

  • fofaapp="H3C-SecPath-运维审计系统" && body="2018"
  • 登录页面
