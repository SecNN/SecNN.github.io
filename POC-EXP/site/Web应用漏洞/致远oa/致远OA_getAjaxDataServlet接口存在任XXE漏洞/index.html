致远OA_getAjaxDataServlet接口存在任XXE漏洞¶

致远互联-OA getAjaxDataServlet 接口处存在XML实体注入漏洞，未经身份认证的攻击者可以利用此漏洞读取系统内部敏感文件，获取敏感信息，使系统处于极不安全的状态。

fofa¶

app="致远互联-OA"
