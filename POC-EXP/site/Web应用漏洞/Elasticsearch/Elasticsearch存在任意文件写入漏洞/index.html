Elasticsearch存在任意文件写入漏洞¶

一、漏洞描述¶

Elasticsearch向使用者提供执行脚本代码的功能，支持mvel, js,groovy,python,和native语言，默认脚本语言为mvel。Elasticsearch存在任意文件写入漏洞

二、影响版本¶

Elasticsearch

三、资产测绘¶
