员工自助平台loginByPassword存在SQL注入漏洞¶

一、漏洞简介¶

广州翰智软件有限公司员工自助平台是指该公司为其员工提供的一种在线平台，旨在提高员工的工作效率、简化人力资源管理流程、增强员工的自主权和满意度。广州翰智软件有限公司员工自助平台 loginByPassword接口处存在SQL注入漏洞，恶意攻击者可能会利用此漏洞修改数据库中的数据，例如添加、删除或修改记录，导致数据损坏或丢失。
