一、漏洞简介¶

广州锦铭泰软件科技有限公司，是一家专业为品牌服饰鞋包企业提供信息化解决方案的高科技企业，该公司开发的F22服装管理软件系统存在信息泄漏漏洞，攻击者最终可利用该漏洞获取数据库账号密码等连接信息。
