海洋CMS后台admin_smtp.php存在远程代码执行漏洞¶

SeaCMS 12.9存在远程代码执行漏洞。该漏洞是由于admin_smtp.php将用户输入的数据未经处理直接拼接写入weixin.php造成的，允许经过身份验证的攻击者利用该漏洞执行任意命令并获取系统权限。

app="海洋CMS"
