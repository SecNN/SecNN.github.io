苹果IOS端IPA签名工具request_post任意文件读取漏洞¶

一、漏洞简介¶

苹果IOS端IPA签名工具request_post任意文件读取漏洞，可能导致敏感信息泄露、数据盗窃及其他安全风险，从而对系统和用户造成严重危害。
