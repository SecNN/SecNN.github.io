月子会所ERP管理云平台SelectUserMangerPrint存在SQL注入漏洞¶

一、漏洞简介¶

月子会ERP管理云平台是由武汉金同方科技有限公司研发团队结合行业月子中心相关企业需求开发的一套综合性管理软件，管控月子中心经营过程中各个环节。月子会所ERP管理云平台SelectUserMangerPrint存在SQL注入漏洞，攻击者可通过该漏洞获取数据库敏感数据。

二、影响版本¶

  • 月子会所ERP管理云平台

三、资产测绘¶

  • fofaproduct="妈妈宝盒-ERP"
  • 登录页面
