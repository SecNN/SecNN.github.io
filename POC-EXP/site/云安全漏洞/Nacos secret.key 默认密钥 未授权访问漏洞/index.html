Nacos secret.key 默认密钥 未授权访问漏洞¶

漏洞描述¶

Alibaba Nacos 使用了固定的secret.key默认密钥，导致攻击者可以构造请求获取敏感信息，导致未授权访问漏洞

漏洞影响¶

Alibaba Nacos <= 2.2.0

网络测绘¶

app="NACOS"

漏洞复现¶

登陆页面

漏洞原因是使用了固定的Key
